It is used to generate the right navigation directory anchor!

## üî® Usage

How to install

```shell
npm install -d vuepress-plugin-anchor-right
```

Use in your project

```js
// docs/.vuepress/config.js

module.exports = {
  ...,
  plugins: [

  ]
};
```
